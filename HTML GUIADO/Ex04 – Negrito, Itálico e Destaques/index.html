Negrito, Italico e Destaques no html

No muno a programação Web, o essencial. ao estudar, percebi que a tag negrito ajuda a destacar as palavras importantes em texto, enquanto a tag italico pode ser usada para nomes obras ou expressoes em outros idiomas

Durante a pratica aprendi a usar o enfase semanticae melhor do que deixar o texto visualmente diferente.Por exemplo quando escrevo algo realmente importante, utilizo a tag para dar destaque ao significado não apenas a aparencia

Em um projeto pessoal, usei varias combinações dessas tags para criar textos mais claros. As tags visuais são uteis em pequenos detalhes, mas as tags semanticasfazem com que o conteudo seja mais compreendido ao buscar leitores
